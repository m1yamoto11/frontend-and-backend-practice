Демо: Адаптивные изображения

Различные техники работы с изображениями

Назад

1. Базовое адаптивное изображение (100% ширины)

Простейший способ сделать изображение адаптивным - использовать CSS свойства:

img {
  max-width: 100%;
  height: auto;
}

Изображение автоматически масштабируется в зависимости от размера контейнера.

2. Элемент <picture> с srcset для разных форматов

Использование современных форматов изображений с fallback на JPEG:

<picture>
  <source type="image/webp" srcset="image.webp">
  <source type="image/avif" srcset="image.avif">
  <img src="image.jpg" alt="Описание">
</picture>

Браузер автоматически выберет наиболее подходящий формат и размер.

3. Разные изображения для десктопа, планшета, мобильных

Использование медиа-запросов для показа разных изображений:

<picture>
  <source media="(min-width: 768px)" srcset="desktop.jpg">
  <source media="(min-width: 576px)" srcset="tablet.jpg">
  <img src="mobile.jpg" alt="Описание">
</picture>
Попробуйте: Измените размер окна браузера, чтобы увидеть, как меняется изображение!

4. Поддержка Retina дисплеев (1x/2x)

Использование srcset для экранов с высокой плотностью пикселей:

<img src="image.jpg"
     srcset="image.jpg 1x, image@2x.jpg 2x"
     alt="Описание">

На экранах с высокой плотностью пикселей браузер загрузит изображение с удвоенным разрешением.

5. Свойства object-fit для стилизации

Демонстрация различных значений object-fit:

object-fit: cover
Сохраняет пропорции, обрезает лишнее
object-fit: contain
Помещает всё изображение в контейнер
object-fit: fill
Растягивает изображение
